# CodeWise AI

CodeWise AI is a powerful and intuitive web application designed to enhance code quality by providing intelligent analysis and improvement suggestions. By leveraging advanced AI, this tool helps developers optimize their code, identify potential bugs, and adhere to best practices, streamlining the development workflow. This project was created by **Aman Kumar**.

---

## Features

- **AI-Powered Code Analysis**: Submit your code snippets and receive insightful suggestions for optimizations, bug fixes, and improvements.
- **Responsive UI**: A clean, modern, and responsive user interface built with Next.js, ShadCN UI, and Tailwind CSS.
- **Real-time Feedback**: Get instant analysis results with a clear loading state indicator.
- **User-Friendly Interface**: A simple two-panel layout allows for easy code input and a clear view of the AI-generated suggestions.
- **Error Handling**: Integrated toast notifications to provide feedback on user actions, such as submitting empty code.
- **Dark Mode**: A sleek and modern dark theme for a comfortable user experience.

---

## Technologies Used

- **Frontend Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Programming Language**: [TypeScript](https://www.typescriptlang.org/)

- **UI Components**: [ShadCN UI](https://ui.shadcn.com/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)

## Getting Started

Follow these instructions to get a local copy of the project up and running.

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/aman179102/CodeWise-Ai.git
    ```

2.  **Navigate to the project directory:**
    ```bash
    cd CodeWise-Ai
    ```

3.  **Install the dependencies:**
    ```bash
    npm install
    ```

### Usage

1.  **Start the development server:**
    ```bash
    npm run dev
    ```

2.  **Open your browser:**
    Navigate to [http://localhost:9002](http://localhost:9002) to see the application in action.

3.  **Analyze your code:**
    - Paste your code into the "Your Code" text area.
    - Click the "Analyze Code" button.
    - Review the AI-generated suggestions that appear in the "AI Suggestions" card.
